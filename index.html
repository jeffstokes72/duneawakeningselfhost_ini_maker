RedBlink Integration Guide

These outputs are fully formatted for RedBlink's Dune: Awakening Self-Host Docker project. To deploy your global settings:

  1. Click Copy File on the blocks below.
  2. Open your RedBlink Web Console in your browser.
  3. Navigate to the Configuration or Settings editor.
  4. Paste the text into the respective editor boxes (or upload as `.ini` files).
  5. Restart your server stack for the Autoscaler to inherit the changes.
