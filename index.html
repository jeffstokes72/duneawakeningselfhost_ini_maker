RedBlink Unified Deployment

RedBlink routes all maps from a single source file, so every instance you add above is compiled into one combined block:

  1. Add an instance for each shard/map you want (e.g. multiple Hagga Basin shards + Deep Desert PvE/PvP).
  2. Click Copy Text on the compiled output below (it contains every partition).
  3. Open your RedBlink Web Console and navigate to the configuration editors.
  4. CRITICAL: Select the UserGame.ini file to edit or import to. (Ignore UserEngine or UserSettings; RedBlink uses UserGame.ini as the unified master file for partitions).
  5. Paste this block at the very bottom of the file and save.
  6. Restart your server stack so the Autoscaler applies the changes.
